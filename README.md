# LeetCoach AI

一键同步你的力扣刷题记录，AI自动分析薄弱点，生成个性化学习计划，告诉你下一步该刷什么。

[主站官网](http://43.142.161.179:8080/)

## 功能特性

- 一键同步力扣（LeetCode）刷题记录
- AI智能分析学习薄弱点
- 生成个性化学习计划
- 智能推荐适合的题目
- 支持力扣中文站（leetcode.cn）和国际站（leetcode.com）

## 安装方法

### 方法一：直接安装.crx文件

1. **下载扩展文件**
   - 从本仓库下载最新版本的 `LeetCoachAI.crx` 文件

2. **安装到Chrome浏览器**
   - 打开Chrome浏览器
   - 进入扩展管理页面：在地址栏输入 `chrome://extensions/`
   - 开启右上角的"开发者模式"
   - 将下载的 `LeetCoachAI.crx` 文件拖拽到扩展管理页面
   - 点击"添加扩展程序"确认安装

### 方法二：从Chrome Web Store安装

- 在Chrome Store里搜索"LeetCoach AI"并安装
- 链接：https://chromewebstore.google.com/detail/leetcoach-ai/bnpkcddgoffcgmdchhngladfkhbbhhbc?authuser=0&hl=en&pli=1

## 使用说明

### 基本使用

1. **打开力扣网站**
   - 访问力扣中文站：https://leetcode.cn/
   - 或访问力扣国际站：https://leetcode.com/
   - 确保已登录你的力扣账号

2. **使用扩展**
   - 点击浏览器工具栏中的LeetCoach AI图标
   - 扩展会自动检测你的登录状态
   - 点击"开始导出"按钮

3. **查看分析结果**
   - 导出完成后，会自动跳转到AI分析页面
   - 在分析页面查看你的学习统计和推荐题目

### 功能说明

- **登录状态检测**：扩展会自动检测你是否已登录力扣账号
- **刷题数据同步**：一键导出所有已完成的题目记录
- **AI分析**：分析你的薄弱知识点和学习模式
- **个性化推荐**：根据分析结果推荐适合的题目

## 支持的网站

- 力扣中文站：https://leetcode.cn/
- 力扣国际站：https://leetcode.com/

## 注意事项

1. **需要登录**：使用前请确保已登录力扣账号
2. **网络连接**：需要稳定的网络连接来同步数据
3. **首次使用**：首次使用可能需要较长时间同步数据
4. **数据安全**：所有数据仅用于个人学习分析，不会泄露给第三方
